# Streamlit

## 概要
このプロジェクトは「倉樫 澄人、N.G.P.、新紀元社」が権利を有する「怪異捜査RPG ツクモツムギ」の二次創作物サイトであり、オンラインセッションなどにおいて使用するPCキャラの能力値画像を出力させるためのアプリケーションです。


## デモ
![デモ画像](preview.png)


## 前提条件
* Python 3.9+
* Streamlit

## インストール
リポジトリをクローンし、必要な依存関係をインストールします。

```bash
# TODO: GitHubに公開後、あなた自身のリポジトリURLに書き換えてください
git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt
```

## 使い方
以下のコマンドを実行してアプリケーションを起動します。

```bash 
# メインのファイル名が app.py ではない場合は、適宜修正してください
streamlit run app.py
```

## 主な機能
* 怪異捜査RPGツクモツムギのオンラインセッションにおいてPCキャラの取得技能と能力値、アップロードしたキャラ画像（任意）を合成して出力します。
